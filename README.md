## Getting Started

First download and install [pip](https://pypi.org/project/pip/)

### Arch Linux
 
 ```bash
 pacman -S python-pip
 ```

 ### Mac OS X

 ```bash
 sudo easy_install pip
 sudo pip install --upgrade pip
 ```

 ## Installation

 ```bash
 pip install imaget
 ```
